# Gudang ERP API

## Menjalankan API
1. Salin `.env.example` menjadi `.env`.
2. Isi kredensial PostgreSQL.
3. Jalankan:
   - `npm install`
   - `npm run start`

## Endpoint
- `GET /health`
- `GET /api/v1/products`
- `POST /api/v1/products`
- `PUT /api/v1/products/:id`
- `DELETE /api/v1/products/:id`
- `GET /api/v1/transactions`
- `POST /api/v1/transactions`
